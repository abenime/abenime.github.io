Project Overview

Tinat is personalized learning platform for Highschool students in ethiopia built with Reactnative and Node.js + express.js to make the learning journey efficient and easier

The platform is built on top oof the current Ministry of Education's Curriculum and includes every course with 800+ questions and materials

Key Features

  • Courses Access Every course for every grade with subject -> chapter -> sub chapter classififcation
  • Exams:Access all the exams and quizes for every subject out there
  • National Exam: All the national exams from 2000 - now
  • Materials: materials from different teachers and videos from youtube
  • Social Network: Connect with other students, share courses and mark

Tech Stack

React Native Expo Node.js Firestore JavaScript Express.js

6 months (From conception to launch)
